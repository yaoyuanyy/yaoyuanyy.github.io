实现代码

声明基础bean: UserInfoParam

UserInfoParam：定义了包含userId,userName的实体bean, 拦截器中只处理@Requestbody中实现了UserInfoParam类的bean。如上文controller中create方法的入参：TemplateParam，继承UserInfoParam

@Data
public class UserInfoParam {

@ApiModelProperty(value = "用户id")
private Long userId;

@ApiModelProperty(value = "用户")
private String userName;

}

定义Filter实现类: UserInfoFilter

HttpServletRequest实现类：CustomHttpServletRequestWrapper

HandlerIntercepter的实现类：CustomInterceptor

定义Configuration class，实现配置。WebMvcConfigurer子类：CustomWebMvcConfigurer

到此，实现功能的代码撸完了。启动spring boot App，就可以curl访问restful接口了
