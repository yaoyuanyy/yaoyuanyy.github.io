如何阅读一本书-读书笔记

2023-02-13 读书 读书 0 评论 字数统计: 2.8k(字) 阅读时长: 9(分)

如何阅读一本书

从没有想到，还有人专门写教人如何读书的书籍。豆瓣评分居然8.3。读完全书后，惊讶就消失了

全书的精华多存在于前两大章。在系统的阐述读书的要点时，冷不丁的蹦出一句很有哲理的句子，给人留下深深的吸引力，想要继续翻阅，获取更多的”惊喜”。

我喜欢阅读纸质书，虽携带不便，但也促使我每天保持固定的时间开始阅读。

全书读完，仿佛以前一些模糊的场景越发的清晰了不少。平时也会读书，此刻，更加笃定：读书一定要主动，主动，再主动。强迫自己要主动。这个想法和动作坚定的存在下来。

作者总结了阅读的三种形态，精准的囊括了阅读的目标

NOTE
阅读的三种形态

  1. 为了娱乐
  2. 获取资讯
  3. 增强理解力

平时的读书是为了什么呢，为了娱乐？为了获取资讯？为了增强理解力？这个是平时最容易忽略的。当阅读文章，小说，书籍等等的那一刻，自问下，以上三种属于哪个，如果是第三个，要提醒自己了：主动阅读。

什么是主动阅读？怎么主动阅读？这本书给我的最重要的理解就是这个答案

读完一本书，要进行总结，进行阅读笔记。也是这本书倡导和要求的，那就从这本《如何阅读一本书》开始吧。

与其说阅读笔记，叫阅读摘要更恰当了。把每章的重要语句摘录出来，成为摘要，便于回顾的同时，能帮助你就更好了

重点来了:看书不是终点，运用书中的知识才是目标

这是一本实用性的书。所以，读完整书，取走书签，合上书本起，重点才刚开始：实践书中的方法，按照作者建议的且你接受的那些方法方式，哪怕一个小点。主动的阅读后，接着是主动的尝试，不断的实践。可能不觉间你的理解力和认知方式会给你很大的惊喜和兴奋

书读的太快或太慢，都一无所获 - 帕斯卡尔

第一篇 阅读的层次

第一章 阅读的活力与艺术

这本书是为增进理解能力的人而写。

短视频给我们的资讯太多了。太多的资讯就如同太少的资讯一样，都是一种对理解力的阻碍。

主动的阅读

双眼停滞，头脑昏睡不叫主动；专心在书本上，专注在阅读上叫主动。专心一致是主动阅读的另一种称呼

读者想获得多少知识完全看他在阅读时多少主动。

NOTE
主动阅读的四个部分

  1. 思考
  2. 观察
  3. 提出问题
  4. 运用想象力

主动阅读核心：在阅读时要提出问题来 - 在阅读的过程中，你自己必须尝试去回答问题

提出什么问题呢？

阅读一本书，一定要提出四个问题：

  1. 整体来说，这本书到底在谈些什么？找出这本书的主题，如何发展的这个主题
  2. 作者细部说了什么，怎么说的？想办法找出主要的想法，论点
  3. 这本书说得有道理吗？是全部有道理，还是部分有道理？
  4. 这本书跟你有什么关系？

阅读越主动，效果越好。这个读者比另一个读者更主动一些，他在阅读世界里面的探索能力就更强一些，收获更多一些，因而也更高明一些。

主动阅读，更容易和作者沟通，获取作者传达给读者的讯息。

》Tips: 倾听是从一位出现在你眼前的老师学习；而阅读却是跟一位缺席的老师学习

为资讯而读，还是为求得理解而读

一个人试着读某样他一开始并不怎么了解的东西。这个东西的水平就是比阅读的人高上一截。这个作者想要表达的东西，能增进阅读着的理解力。这样的学习是理解更多的事情，而不是记住更多的资讯。

第二章 阅读的层次

1.基础阅读：认识字词，通常小学时就完成了
2.检视阅读：一定时间内，抓出一本书的重点；这本书在谈什么；这本书包含哪些部分；这本书的架构如何
3.分析阅读：全盘的阅读，完整的阅读，优质的阅读，最好的阅读方式；分析阅读擅长于寻求理解
4.主题阅读：所有阅读中最有收获的，获益良多

第三章 阅读的第一个层次：基础阅读

随着上学，自然就会的

第四章 阅读的第二个层次：检视阅读

检视阅读也叫略读。分两种方式

  1. 粗读
  2. 粗浅的读

你可以把自己想成是一个侦探，在找寻一本书的主题或思想的线索

NOTE: 如果不能够集中精神来阅读，就没法进行检视阅读

粗读

粗读一本书的步骤：

  1. 先看书名页，然后如果有序就先看序：对这本书的主题有个概念
  2. 研究目录页：对这本书的基本架构有个概括性的理解。就像旅行前，先看下地图一样。
  3. 挑几个看着跟主题息息相关的篇章来看
  4. 把书打开，东翻翻西翻翻，念个一两段，连续读几页，但不要太多
  5. 如果有后记，一本书最后结尾的两三页页不可忽视

粗浅的读

从头到尾先读一遍，碰到不懂的地方不要停下来查询或思索。集中精神，继续读下去，将全书读完

很实用的方法，下文还是重复
检视阅读一定要提问和回答三个问题：

  1. 这是什么样的一本书？
  2. 整本书在谈的是什么？
  3. 作者是借着怎样的架构，来发展他的观点或陈述他对这个主题的理解？

第五章 如何做一个自我要求的读者

  • 做一个自我要求的读者，两方面
  1. 主动阅读：提出问题，尝试回答
  2. 培养阅读的习惯：习惯是第二天性

如果你的阅读目标是获得利益 - 你就得保持清醒

  • 阅读一本书，一定要提出四个问题：
  1. 整体来说，这本书到底在谈些什么？找出这本书的主题，如何发展的这个主题
  2. 作者细部说了什么，怎么说的？想办法找出主要的想法，论点
  3. 这本书说得有道理吗？是全部有道理，还是部分有道理？
  4. 这本书跟你有什么关系？
  • 书上做笔记
    阅读时，将你的思考，你的感想写下来，能帮助你记住作者的思想。

Tips 作笔记最好的地方是目录页，或是书名页

作笔记方法：

  1. 关键字词画底线 - 我常用的
  2. 做符号
  3. 在空白处记下其他的页码
  4. 在书页的空白处做笔记 - 我常用的

不同阅读层次做不同类型的笔记
检视阅读 - 结构笔记(架构笔记)
分析阅读 - 概念笔记
主题阅读 - 辨证笔记

第一篇 阅读的第三个层次：分析阅读

透视一本书

分析阅读：用一双X光的透视眼来看这本书，这是你了解掌握一本书的基础

分析阅读的阶段和规则：

  • 第一阶段：掌握大纲(这本书在谈些什么)
  1. 依照书本的种类与主题作分类 - 将书分类
  2. 使用一个单一的句子，或最多一小段文字来叙述整本书的内容 - 说出整本书的大意
  3. 将书中重要篇章列举出来，说明它们如何按照顺序组成一个整体的架构 - 列出重要部分，拟定大纲
  4. 找出作者在问的问题，或作者想要解决的问题 - 发现作者的意图

揭开外衣，撕去肌肉，得到骨架。观察一个人，了解一件事，都应该这么做

  • 第二阶段：诠释内容(这本书的详细内容是什么)
  1. 找出关键字，与作者达成共识 - 共识
  2. 将一本书中最重要的句子圈出来，找出其中的主旨：最重要的句子就是阐述作者判断的部分 - 主旨
  3. 从相关文句的关联中，设法架构处一本书的基本论述 - 论述
  4. 确定作者已经解决了什么问题，还有哪些是没有解决的 - 问题

还记得开头时说阅读一本书要回答四个问题吗：这本书的详细内容是什么？如何叙述的？做完阶段二的四个规则就可以回答了。当你跟作者达成共识，找出他的关键主旨与论述，分辨出如何解决他所面对的问题，你就会知道这本书要说的是什么了。

写作准则：一个作品应该有整体感，清楚明白，前后连贯

责任是所有正确行为的基础 - 康德

阅读的一部分本质就是被困惑，而且知道自己被困惑

怀疑是智慧的开始

  • 第三阶段-公正地判断一本书
  1. 在你说出：“我同意”，“我不同意”之前，你一定要能肯定的说：“我了解了”
  2. 当不同意作者的观点时，要理性的表达自己的意见，不要无理的辩驳或争论
  3. 尊重知识与个人观点的不同，任何判断，都要有理论依据

三个阶段，循序渐进。如果只想检视阅读，第一阶段的四个规则就可以了。如果想要分析阅读，十一个规则恐怕都要用上了。但是，必须强调的是，这些规则是一个理想化的阅读。只是衡量阅读层次的理想标准。如果你想做个好读者，那么就要达到相应的阅读层次

后面的章节都是这些规则的变种。应对不同的读物，稍许的不同。所以就不进行阅读笔记了。

值得一题的是主题阅读，之前我们的目标是一本书。现在是一个主题，却对应的多本书，对应着多本书中的几个章节。自然的想法是只有阅读了那些本书，针对一个主题阅读时，才能想到那些书籍。所以，关键还是要阅读每一本书。

总结

阅读一本值得的书籍时，主动的阅读。对应着碎片化的流行，我们要系统的获取知识。系统的获取还有别的方式比阅读一本书更系统吗
