prompt-结构化提示词-LangGPT

2025-07-20 AI prompt 0 评论 字数统计: 3.6k(字) 阅读时长: 13(分)

背景

在我学习AI编程和AI问答的时候，无论是使用deepseek 还是 chatgpt，发现喂给他们的提示词非常关键，提示词给的准，LLM给的越准。

看了很多prompt的文章，LLM在快速的发展，网上更多的是重复的知识和文章，掐头去尾了。搜索有价值的东西越来越难了。

先是看到了许多CRISPE，BREKE等框架的知识，最后，看到了最想学习的LangGPT。

LangGPT - 结构化Prompt

LangGPT - language of gpt：面向LLM打模型的自然语言编程框架。

Prompt(提示)设计规则

  1. 具有规范化的格式
  2. 可扩展
  3. 要求必须具体，清晰，完整
  4. 语言要灵活

一个 Prompt template 包含以下模块

  1. Profile：指出了用户对LLM在角色方面的要求，包括个人简介、人物肖像等。
  2. Goal：列出了用户想要实现的目标，这就是LLM需要完成的目标。
  3. Skill：用于向LLM表明他们拥有的技能。经过工具学习的LLM，可以指导他们调用工具，更准确地执行任务。
  4. Constraint：表示约束或突出，即不允许LLM超出的范围以及生成回复时必须满足的要求等。
  5. Rules： 角色必须遵守的规则，通常是角色必须做的或者禁止做的事情，比如 “不许打破角色设定” 等规则。
  6. Example：给出了输入-输出对作为供LLM学习的例子。
  7. Workflow：指示了执行任务时的工作流程，当任务需求比较复杂时，往往需要实例化这个模块。
  8. Suggestion：包括对LLM的建议和行为规划。该模块重点列出常见场景，并告诉LLM在此类情况下可以采取的行为或应对措施
  9. Background：表示LLM在执行任务时需要具备的背景信息和记忆。
  10. Style：限定了LLM生成回复的风格。
  11. Output Format：定义了LLM的输出格式。指定输出格式可以提高某些任务中结果提取的效率和准确。
  12. Initialization：即初始化，以告知LLM即将开始对话。有时，该模块中还会给出指定的第一句话。

Prompt template

ChatGPT 很擅长角色扮演，只要提供角色说明，角色行为，技能等描述，就能做出很符合角色的行为。

因此 LangGPT 设计了模板让 ChatGPT 更好的理解用户意图，并相应提供了一套角色设计方法。

LangGPT 提供了 Prompt 编写模板，套用模板即可快速编写高质量 Prompts。

Role 模板

Role 模板主要包含四部分内容:

  • Profile 角色的简历: 角色描述，角色特点，角色技能以及你想要的其他角色特性。
  • Rules 角色必须遵守的规则，通常是角色必须做的或者禁止做的事情，比如 “不许打破角色设定” 等规则。
  • Workflow 角色的工作流，需要用户提供怎样的输入，角色如何响应用户。
  • Initialization 按照 Role 模板的配置初始化角色，大部分时候使用模板默认内容即可

Prompt template 使用步骤

  1. Role 设置角色名：将 Role：Your_Role_Name 中的 Your_Role_Name 替换为你的角色名

  2. Profile 编写角色简历：

    • 设置语言，Language 设置为 中文 或者 English 等其他语言, 用目标语言表达为佳
    • Description 后面简单描述角色

  3. Skill 部分添加角色技能：可以设置多个技能，技能下分点提供技能描述

  4. Rules 设定规则：添加角色必须遵守的规则，通常是角色必须做的或者禁止做的事情，比如 “Don’t break character under any circumstance.” “禁止出戏” 等规则

  5. Workflow 设定工作流：角色如何与用户交互，需要用户提供怎样的输入，角色如何响应用户。

  6. Initialization 初始化角色：Role 模板依据模板内容对角色进行设定，一般不需要修改。

  7. 将编写好的 Role 模板内容复制到 ChatGPT 对话框（or API）愉快使用~

高级用法

变量为 Prompt 的编写带来了很大的灵活性。使用变量可以方便的引用角色内容，设置和更改角色属性。

Role 模板里的Initialization部分则大量使用的了变量：

  • 变量，指代了整个 Role 角色的内容。
  • 变量，指代了 ## Rules 一节的规则
  • 变量，指代了 Language 字段的值

如何写好结构化Prompt

Prompt = 理解LLM + Know-how + 逻辑清晰

  • 理解LLM：了解每个模型的特点和能力，是构建有效 Prompt 的基础。

  • Know-how：对于一个领域、一个行业，一个细分场景的理解到底有多深？

  • 逻辑清晰：逻辑这个层面上，也就是表达层面上，一个人是不是能够把一个复杂的事情给说清楚了

结构化 Prompt 对不同模型的适用性

结构化 Prompt 编写对模型基础能力有一定要求，要求模型本身具有较好的指令遵循、结构识别分析能力。从实践来看，GPT-4 是最佳选择， Claude 模型能力次之， GPT-3.5 勉强可用。依据笔者实践和身边朋友使用的反馈来看，在 GPT-4 和 Claude 模型上的表现情况都不错， GPT-3.5 则存在表现不稳定现象。
